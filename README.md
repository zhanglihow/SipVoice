# SipVoice

该demo基于[https://github.com/VoiSmart/pjsip-android](https://github.com/VoiSmart/pjsip-android)
项目实现语音对讲对讲功能。
还有很多关于 pjsip 的功能和API请移步[https://github.com/VoiSmart/pjsip-android](pjsip-android) 。
  
  
  
    
demo中，实现了基本的语音对讲的功能，拨号，接听，静音，外放，计时等...
